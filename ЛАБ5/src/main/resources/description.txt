    2. Если пользователь ввел, правильные данные, он аутентифицирован.
        2.1 В этом случае пусть создается объект User, cодержащий логин, пароль.
        2.2 сли данные не правильные, то пусть создается объект User, с пустыми значениями.
б) просмотр списка каталогов товара CATALOG
в) просмотр списка товаров определенного каталога, отсортированного по заданному критерию
    ITEM

    CART
